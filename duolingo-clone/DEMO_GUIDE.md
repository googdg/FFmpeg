## 🎮 演示流程

### 完整用户体验流程

1. **访问首页** (http://localhost:3000)
   - 查看多邻国风格的欢迎界面
   - 点击 "Get Started" 注册新账户

2. **注册账户**
   - 填写邮箱: `demo@example.com`
   - 用户名: `demo_user`
   - 密码: `Demo123!` (观察密码强度条)
   - 选择母语: `中文 (Chinese)`
   - 点击 "Create Account"

3. **自动跳转到学习页面**
   - 查看欢迎消息和课程介绍
   - 观察顶部的课程进度条
   - 查看技能树结构

4. **开始第一个技能**
   - 点击 "Greetings" 技能节点 (绿色圆圈)
   - 查看课程预览模态框
   - 点击 "Start Lesson" 开始学习

5. **完成练习题**
   - **选择题**: "How do you say 'Hello' in English?" → 选择 "Hello"
   - **翻译题**: "Translate: 你好" → 输入 "Hello"
   - **听力题**: 选择正确的听到内容
   - **填空题**: "Fill in the blank: _____ morning!" → 输入 "Good"
   - **排序题**: 将单词排列成正确顺序

6. **查看学习成果**
   - 观察XP增加动画
   - 查看课程完成统计
   - 注意顶部导航栏的统计更新

7. **探索其他功能**
   - 点击顶部的 "Stories", "Leaderboard", "Shop" (占位页面)
   - 查看用户头像和统计信息
   - 测试移动端响应式设计
