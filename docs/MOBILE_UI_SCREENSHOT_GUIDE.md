# 📱 移动端UI截图指南

## 📋 截图概述

**目标设备**: 移动端 (375px 宽度，iPhone SE尺寸)  
**截图用途**: 为UI设计师提供移动端界面设计参考  
**截图格式**: PNG格式，高清晰度  

## 🚀 快速开始

### 1. 启动本地服务器
```bash
python3 -m http.server 3000
```

### 2. 设置浏览器
- 打开Chrome浏览器
- 按F12打开开发者工具
- 点击设备模拟器图标 📱
- 选择"iPhone SE" (375 × 667)

### 3. 访问页面
```
http://localhost:3000/aoyou-youtube-style.html
```

## 🎨 移动端设计要点

### 布局特点
```
┌─────────────────────────┐
│ 🏥 奥友医学    [头像]   │  ← 导航栏 (56px高)
├─────────────────────────┤
│ ← [分类标签滑动区域] →  │  ← 分类栏 (48px高)
├─────────────────────────┤
│ ┌─────────────────────┐ │
│ │                     │ │
│ │    视频缩略图        │ │  ← 单列视频布局
│ │                     │ │
│ │ [头像] 视频标题      │ │
│ │        讲师信息      │ │
│ └─────────────────────┘ │
│ ┌─────────────────────┐ │
│ │    下一个视频        │ │
│ └─────────────────────┘ │
└─────────────────────────┘
```

### 关键尺寸
- **屏幕宽度**: 375px
- **导航栏高度**: 56px
- **分类栏高度**: 48px
- **视频卡片**: 全宽，16px左右边距
- **最小触摸目标**: 44px × 44px

### 移动端特色
- **隐藏搜索框**: 移动端导航栏不显示搜索框
- **横向滚动**: 分类标签支持左右滑动
- **单列布局**: 视频采用单列垂直排列
- **触摸优化**: 按钮和链接适合手指点击

## 🎯 给UI设计师的移动端设计建议

### 1. 触摸友好设计
- 按钮最小尺寸44px × 44px
- 增加按钮间距，避免误触
- 使用适合手指操作的控件

### 2. 内容优先
- 简化导航，隐藏次要功能
- 突出核心内容（视频）
- 优化信息层次

### 3. 滑动交互
- 分类标签横向滑动
- 视频列表垂直滚动
- 添加滑动指示器

### 4. 性能优化
- 图片懒加载
- 减少动画复杂度
- 优化字体加载

### 5. 医学专业感
- 保持专业的视觉风格
- 使用医学相关图标
- 体现权威性和可信度
