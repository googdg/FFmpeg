## 📊 数据案例

系统包含5个典型案例，覆盖所有风险等级：

| 案例 | 风险等级 | 评分 | 决策建议 | 特点 |
|------|---------|------|---------|------|
| 案例1 | A级 | 850 | 批准 | 优质客户，资质优秀 |
| 案例2 | B级 | 720 | 有条件批准 | 良好客户，负债率偏高 |
| 案例3 | C级 | 580 | 有条件批准 | 一般客户，存在逾期 |
| 案例4 | D级 | 420 | 建议拒绝 | 关注客户，风险较高 |
| 案例5 | E级 | 280 | 立即拒绝 | 失信客户，风险极高 |

## 📱 使用方式

### 推荐方式：使用本地服务器
```bash
# 使用启动脚本
cd vw-credit-ai-report
./start.sh

# 访问案例测试页（推荐入口）
http://localhost:8000/test-cases.html

# 或直接访问报告页
http://localhost:8000/final.html
```

### 快速测试
```bash
# 运行功能测试脚本
./test-all.sh

# 查看快速开始指南
cat QUICK_START.md
```

### 直接访问特定案例
```bash
# 使用URL参数
http://localhost:8000/final.html?case=case1  # A级客户
http://localhost:8000/final.html?case=case5  # E级客户
```
