# 大众汽车金融 - AI信审报告系统 (MVP版本)

## 📋 项目简介

这是一个为大众汽车金融服务公司开发的AI驱动的信审报告系统MVP版本。系统能够自动生成符合行业规范的"一页纸信审报告"，帮助信审员快速做出决策。

## ✨ 核心功能

- ✅ **一页纸报告展示** - 清晰简洁的报告布局
- ✅ **AI风险评估** - 多维度风险评分和等级(A/B/C/D/E五级)
- ✅ **RAG智能汇总** - 自动汇总财报、第三方数据和舆情
- ✅ **AI建议审核方向** - 智能生成审核重点
- ✅ **AI辅助提问** - 自动生成建议问题清单
- ✅ **风险警示模块** - 分级显示高/中/低风险警示
- ✅ **多案例演示** - 5个完整的不同风险等级案例
- ✅ **案例切换** - 支持在报告页面快速切换案例
- ✅ **案例测试页** - 专门的案例浏览和测试页面
- ✅ **PDF导出** - 支持报告导出为PDF
- ✅ **打印优化** - 优化的打印样式

## 🎯 使用说明

### 1. 浏览案例

**方式A: 使用案例测试页(推荐)**
- 访问 `test-cases.html` 查看所有案例概览
- 点击任意案例卡片查看详细报告
- 包含案例统计和快速导航

**方式B: 直接访问报告页**
- 访问 `final.html` 查看报告
- 使用顶部下拉菜单切换案例
- 支持URL参数: `final.html?case=case1`

### 2. 案例说明

系统包含5个完整的测试案例：

| 案例 | 风险等级 | 评分 | 决策 | 特点 |
|------|---------|------|------|------|
| 案例1 | A级优质 | 850 | ✅ 批准 | 收入稳定,无逾期,低负债率 |
| 案例2 | B级良好 | 720 | ⚠️ 有条件批准 | 负债率偏高,需额外审核 |
| 案例3 | C级一般 | 580 | ⚠️ 有条件批准 | 有逾期记录,需担保 |
| 案例4 | D级关注 | 420 | ❌ 拒绝 | 多次逾期,涉及诉讼 |
| 案例5 | E级拒绝 | 280 | ❌ 立即拒绝 | 失信名单,严重违约 |

### 3. 报告内容

每个报告包含以下模块：
- **客户概览** - 基本信息和申请详情
- **风险评估** - 综合评分和多维度雷达图
- **关键指标** - DSR、LTV、征信分等核心指标
- **RAG智能汇总** - 财报、第三方数据、舆情监测
- **AI建议审核方向** - 智能生成的审核重点
- **风险警示** - 分级显示高/中/低风险点
- **审批建议** - AI推荐的决策和条件
- **AI辅助提问** - 建议的问题清单
- **RAG智能汇总** - 财报、第三方数据、舆情
- **AI建议审核方向** - 审核重点提示
- **审批建议** - AI给出的决策建议
- **AI辅助提问** - 建议问题清单（可展开）
- **风险提示** - 风险因素列表
- **决策依据** - 数据来源和模型信息

### 3. 导出报告

- **打印** - 点击"打印报告"按钮，使用浏览器打印功能
- **PDF** - 点击"导出PDF"按钮，自动生成PDF文件

## 🔧 自定义配置

### 修改品牌色

编辑 `css/main.css` 中的CSS变量：

```css
:root {
    --primary-color: #001e50;    /* 主色 */
    --secondary-color: #00b0f0;  /* 辅助色 */
    --accent-color: #0066cc;     /* 强调色 */
}
```

### 添加新案例

编辑 `js/data.js`，在 `mockCases` 对象中添加新案例：

```javascript
const mockCases = {
    case6: {
        customer: { /* 客户信息 */ },
        riskAssessment: { /* 风险评估 */ },
        // ... 其他数据
    }
};
```

## 📝 MVP限制说明

当前MVP版本的限制：

1. **数据为模拟数据** - 使用静态JSON，不连接真实数据源
2. **无后端服务** - 纯前端实现，无需服务器
3. **无用户认证** - 无登录功能
4. **AI功能为模拟** - RAG和AI建议为预设文本
