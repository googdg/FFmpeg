# 🚀 快速开始指南

## 1分钟快速体验

### 步骤1: 启动服务器

```bash
cd vw-credit-ai-report
./start.sh
```

服务器将在 `http://localhost:8000` 启动

### 步骤2: 访问系统

打开浏览器,访问以下任一页面:

#### 🎯 推荐入口: 案例测试页
```
http://localhost:8000/test-cases.html
```
- 查看所有5个案例的概览
- 快速导航到任意案例报告
- 包含案例统计信息

#### 📄 报告页面
```
http://localhost:8000/final.html
```
- 查看完整的AI信审报告
- 使用顶部下拉菜单切换案例
- 支持PDF导出和打印

#### 🎨 Vue版本 (功能相同)
```
http://localhost:8000/index.html
```
- 使用Vue.js实现的版本
- 功能与final.html完全相同

## 🎯 核心功能测试

### 1. 案例切换
- 在报告页面顶部使用下拉菜单
- 或访问 test-cases.html 点击案例卡片

### 2. PDF导出
- 点击报告页面右上角的"📄 导出PDF"按钮
- 系统会自动生成并下载PDF文件

### 3. 打印报告
- 点击"🖨️ 打印"按钮
- 或使用快捷键 Ctrl/Cmd + P
- 系统已优化打印样式

### 4. 查看不同模块
每个报告包含:
- 👤 客户概览
- ⚠️ 风险评估 (含雷达图)
- 🤖 RAG智能汇总
- 🎯 AI建议审核方向
- ⚠️ 风险警示
- ✅ 审批建议
- 💬 AI辅助提问

### 常见问题

**Q: 页面显示空白?**
A: 确保使用HTTP服务器访问,不要直接打开HTML文件

**Q: 案例切换没反应?**
A: 检查浏览器控制台是否有JavaScript错误

**Q: PDF导出失败?**
A: 确保网络连接正常(需要加载html2pdf.js库)

**Q: 如何停止服务器?**
A: 在终端按 Ctrl+C
