# 📸 移动端UI截图操作指南

## 🎯 目标

为UI设计师提供7个关键的移动端界面截图，用于专业UI设计参考。

## 🛠️ 准备工作

### 1. 启动本地服务器
```bash
cd 项目目录
python3 -m http.server 3000
```

### 2. 设置Chrome浏览器
1. 打开Chrome浏览器
2. 按 `F12` 打开开发者工具
3. 点击设备模拟器图标 📱 (或按 `Ctrl+Shift+M`)
4. 在设备下拉菜单中选择 **"iPhone SE"**
5. 确认尺寸显示为 **375 × 667**

## 🚀 快速截图技巧

### Chrome快捷键
- `F12` - 打开开发者工具
- `Ctrl+Shift+M` - 切换设备模拟器
- `Ctrl+Shift+P` - 打开命令面板
- 输入 "screenshot" 查看截图选项

### 截图选项
- **Capture screenshot** - 当前视口截图
- **Capture full size screenshot** - 完整页面截图
- **Capture node screenshot** - 选中元素截图

## ❓ 常见问题

### Q: 邀请码弹窗不显示？
A: 可能已经验证过了，在控制台输入 `resetVideoAccess()` 重置状态。

### Q: 移动端布局不正确？
A: 确认开发者工具中设备选择为 "iPhone SE"，宽度显示375px。

### Q: 截图模糊？
A: 确保浏览器缩放比例为100%，使用Chrome的内置截图功能。

### Q: 页面加载不完整？
A: 等待页面完全加载后再截图，确保所有图片和样式都已加载。

## 📞 技术支持

如果遇到问题：
1. 检查本地服务器是否正常运行
2. 确认浏览器设置正确
3. 清除浏览器缓存后重试
4. 使用隐身模式测试
